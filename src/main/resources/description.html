Tree Gen

Generate ASCII tree representations of project directories

Features:

  • Generate directory trees directly from the project view
  • Customizable ignore patterns for excluding files and directories
  • Option to show hidden files
  • Option to ignore excluded directories
  • Option to file and directory sizes
  • Option to copy to clipboard or save to file

Usage:

  1. Right-click on any directory in the Project view
  2. Select "Generate Directory Tree"
  3. Configure options in the dialog
