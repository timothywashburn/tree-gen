Directory Tree Generator

Generate ASCII tree representations of your project directories with customizable options

Features:

  • Generate directory trees directly from the project view
  • Customizable ignore patterns for excluding files and directories
  • Option to show or hide hidden files
  • Include file and directory sizes
  • Copy tree to clipboard or save to file

Usage:

  1. Right-click on any directory in the Project view
  2. Select "Generate Directory Tree"
  3. Configure options in the dialog:
    • Show hidden files
    • Show file/folder sizes
    • Add custom ignore patterns
    • Generate output file
